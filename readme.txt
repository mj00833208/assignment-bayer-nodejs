#Installation

 * Make sure following versions are installed

   - NodeJS - Node versions

   - Express - Express versions

 * Clone codebase
  
   - Run in project directory > git clone  path

   - Run in project directory > git checkout current branch

 * Once clone is successful, to install required modules

   - Run in project directory > npm install 
 * Run local Node JS server, once npm install complete

   - Run in project directory > npm run start-local
